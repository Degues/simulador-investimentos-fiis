Simulador de Investimentos em FIIs

Sobre o projeto:

- Este projeto foi desenvolvido em Excel com o objetivo de simular investimentos em Fundos de Investimento Imobiliário (FIIs). 
A planilha permite visualizar cenários de investimento e acompanhar a evolução do patrimônio ao longo do tempo, auxiliando na análise e no planejamento financeiro.

Objetivos:

- Praticar conceitos de Excel.
- Desenvolver uma ferramenta de simulação financeira.
- Aplicar fórmulas, cálculos e organização de dados.
- Criar um projeto para compor portfólio de estudos.

Ferramentas utilizadas:

- Microsoft Excel

Arquivos:

- Simulador de Investimentos em FIIs (.xlsx)
